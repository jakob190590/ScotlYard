This package contains the ScotlYard core interfaces and default implementations. The GameState defines a read-only interface to access the complete state and history of the game. Game extends the GameState to allow modification of the GameState. Moves are part of the history. A Move object describes a move of a player. A GameState is the model when used in MVC architecture. So it implements methods to add and remove listeners. There are four listener interfaces for different kinds of changes in the GameState.
